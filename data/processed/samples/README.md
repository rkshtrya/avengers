# Sample Files Guide

This folder contains small CSV samples created from the raw data.

Use these when you want to inspect columns and example rows without opening the full dataset.

## How To Use Samples

1. Open a sample CSV in GitHub.
2. Look at the column names.
3. Look for possible join fields such as APN, address, city, ZIP, coordinates, geometry fields, tract IDs, school IDs, or permit numbers.
4. Compare what you find with `../../source_registry.csv`.
5. If something important is missing, write the question back into `docs/` or `reports/`.

## Important Note

Samples are for review only.

Do not use sample files to make final counts or product decisions. Use the notebooks or scripts for that.
